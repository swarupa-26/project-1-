# project-1-

Stock Portfolio Tracker

A simple and interactive Python command-line application that allows users to track their stock investments.
It calculates the total portfolio value using hardcoded stock prices and gives the option to save a summary to a text file.

 Overview

This program helps you:

Input the number of shares you own for specific stocks

Calculate the total value of your investment portfolio

Display a detailed summary of your holdings

Save your portfolio summary to a text file

It’s a great beginner-friendly example of using Python dictionaries, loops, and file handling together.

 Features

 Portfolio Value Calculation — Computes total investment based on share quantities and prices

 Detailed Summary — Displays each stock’s price, quantity, and value

 Save to File — Optionally exports your portfolio summary as a .txt file

 Error Handling — Handles invalid stock symbols, negative quantities, and non-numeric inputs gracefully

 Requirements

Python 3.7 or higher

No external libraries are required — this script only uses Python’s built-in modules

 How to Run

Clone or download this project:

git clone https://github.com/yourusername/stock-portfolio-tracker.git
cd stock-portfolio-tracker


Run the program:

python portfolio_tracker.py


Follow the on-screen prompts:

Enter a valid stock symbol (e.g., AAPL, TSLA)

Enter the number of shares you own

Type done when finished

Choose whether to save your summary
